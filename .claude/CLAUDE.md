# Booking App

## Project Overview
A booking application built with React 19, TypeScript, and Vite 7.

## Tech Stack
- **Framework:** React 19 with TypeScript
- **Build Tool:** Vite 7
- **Linting:** ESLint (flat config) with typescript-eslint, react-hooks, react-refresh plugins

## Commands
- `npm run dev` — Start dev server
- `npm run build` — Type-check and build for production
- `npm run lint` — Run ESLint
- `npm run preview` — Preview production build

## Project Structure
- `src/main.tsx` — App entry point
- `src/App.tsx` — Root component
- `src/index.css` — Global styles
- `src/App.css` — App component styles

## Conventions
- Use TypeScript for all source files (.ts/.tsx)
- Use functional components with hooks
- CSS files colocated with their components
